# Как пользоваться промптами

1. Открой `_base.md` — это неизменная часть описания персонажа Хӏарпик, которая
   должна попадать в каждый запрос к генератору изображений (сохраняет консистентность
   стиля, палитры, пропорций между всеми 20 стикерами).
2. Возьми файл нужного стикера (например `01_barkala_smile.md`) — в нём указана
   специфика: эмоция, поза, жест рук, доп. пропсы и текст подписи.
3. Склей промпт: текст из `_base.md` + текст из `Prompt (эмоция/поза/пропсы)` конкретного
   файла. Подпись ("Caption overlay") добавляется либо в сам промпт, либо накладывается
   отдельно в графредакторе после генерации — второй вариант надёжнее для читаемости текста.
4. Генерируй в 2048×2048, экспортируй в 512×512 PNG с прозрачным фоном → кладём в `../png/`.
5. При необходимости — векторизация в `../svg/`, финальная сборка пака — в `../exports/`.
