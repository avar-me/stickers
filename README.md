# Хӏарпик — стикерпак avar.me

Маскот проекта [avar.me](https://avar.me) — ожившая книга аварского словаря. Дружелюбный,
немного ироничный и очень умный персонаж. На левой странице книги — «АВ», на правой — «РУ».

- **Сайт:** [stickers.avar.me](https://stickers.avar.me)
- **Стикерпак в Telegram:** [t.me/addstickers/AvarMe](https://t.me/addstickers/AvarMe)

Полное описание концепции, характера персонажа и состава наборов — в [`plan.md`](plan.md).

## Структура репозитория

```
stickers/
├── plan.md            # исходное ТЗ: концепция, характер, состав наборов
├── concepts/
│   └── character.md   # утверждённый референс персонажа (стиль, палитра, детали)
├── prompts/            # готовые промпты для генерации каждого стикера
│   ├── _base.md        # общая часть (вставлена в каждый промпт)
│   └── NN_*.md          # промпт конкретного стикера (base + эмоция/поза/подпись)
├── png/final/           # мастер-файлы стикеров (высокое разрешение, как сгенерированы)
├── exports/              # финальные PNG 512×512 с прозрачным фоном — то, что грузится в Telegram
└── docs/                 # статический сайт stickers.avar.me (GitHub Pages)
```

## Как добавить новый стикер

1. Скопировать `prompts/_base.md` + написать специфику (эмоция/поза/пропсы/подпись) —
   см. любой файл `prompts/NN_*.md` как образец.
2. Сгенерировать картинку, **скачать оригинальный файл** (не скриншот превью — иначе
   вместо прозрачного фона запечётся шахматный узор).
3. Проверить прозрачность программно (альфа-канал по углам должен быть 0), а не на глаз.
4. Положить мастер-файл в `png/final/`, обрезать/отмасштабировать в 512×512 → `exports/`.
5. Скопировать в `docs/img/` и добавить в галерею `docs/index.html`, если стикер входит
   в публичный набор на сайте.
6. Загрузить в Telegram через [@Stickers](https://t.me/addstickers).

## Сайт

`docs/` — статический сайт на чистом HTML/CSS (без сборки), в дизайн-системе,
согласованной с `avar.me` / `dev.avar.me` / `forms.avar.me`. Публикуется через
GitHub Actions (`.github/workflows/deploy.yml`) на каждый push в `main`.
